# Product

## Register

product

## Users

Idle and incremental game players using a mobile-first web app. They are watching AUTO mining run, checking resources, choosing rewards, and unlocking permanent Mining Core nodes during short sessions.

## Product Purpose

Endless Mine is a dark cozy first-person idle mining game. The MVP must make the 3x3 mining face, AUTO pickaxe strikes, resource gain, reward encounters, and full Mining Core progression playable in a Web PWA.

## Brand Personality

Dark, tactile, cozy. The game should feel like a lantern-lit mine with readable systems and satisfying feedback, not a generic dashboard or a realistic photo set.

## Anti-references

Do not make the interface look like a shop, gacha app, ad-driven mobile game, generic icon pack, or realistic photo set. Avoid decorative SaaS tropes and keep the mining screen focused on play.

## Design Principles

1. Show the action: every mined hit must visibly connect the pickaxe, target cell, impact, cracks, dust, and resource feedback.
2. Keep systems readable: resources, AUTO state, current target, rewards, and node unlock requirements should be legible at 480x720.
3. Respect generated progression data: all Mining Core nodes, parent links, costs, and effect keys stay visible and wired.
4. Tune in the app: balance and pacing values belong in the Debug Panel while the MVP is still being shaped.
5. Stay web-first and portable: isolate platform APIs so later Tauri work does not invade game systems.

## Accessibility & Inclusion

Support reduced motion by simplifying animation rather than removing feedback. Use readable contrast, large touch targets, semantic buttons, and debug-visible settings for motion, sound, and visual asset modes.
